### Getting started

## Installation via Farmer (Recommended)

1. Install [Farmer](https://github.com/jonasarensmann/farmer)

2. Create a new project and add Aurora

```bash
farmer init aurora-example
farmer add aurora
```

3. Add the example code to src/main.luau

```lua
local aurora = require("../crops/aurora")

-- GET request to /
aurora.get("/", function(req, res)
    res.send("Hello World")
end)

aurora.createServer() -- runs the server on port 8080
```

4. Run the server

```bash
lune run src/main.luau
```

## Installation via Git

1. Clone the Repository at the root of your project

```bash
git clone https://github.com/jonasarensmann/aurora.git
```

2. Create main.luau in your project root and add the example code

```lua
local aurora = require("aurora/src/main")

aurora.get("/", function(req, res)
    res.send("Hello World")
end)

aurora.createServer({
    port = 8080,
    debug = true,
})
```

3. Run the server

```bash
lune run main.luau
```
